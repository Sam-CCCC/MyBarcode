BarcodePro 隐私政策

您的隐私对我们至关重要

最近更新日期：2024年3月

感谢您使用 BarcodePro。本隐私政策旨在帮助您了解我们如何处理您的信息。我们承诺保护您的隐私，并确保提供透明的服务体验。

1. 信息收集

BarcodePro 是一款本地运行的工具应用。

  • 个人数据： 我们不收集、不存储也不共享任何个人身份信息。
  • 本地存储： 您的配置参数（如字号、比例、语言设置）存储在您设备浏览器的本地存储（LocalStorage）中。这些数据不会上传到任何服务器。
  • 文件访问： 当您使用 Excel 导入功能时，数据仅在您的浏览器内存中处理，处理完毕后即刻释放。

2. 权限说明

应用可能会申请以下权限，仅用于实现特定功能：

  • 存储权限： 用于保存您导出的条码配置 JSON 文件或保存生成的图片。
  • 摄像头权限：（如果未来加入扫描功能）仅用于条码识别。

3. 第三方服务

本应用托管在 GitHub Pages 上。GitHub 可能会收集某些标准日志信息（如 IP 地址），此类收集受 GitHub 隐私政策约束。我们本身不集成任何第三方广告跟踪或分析插件。

4. 数据安全性

由于所有数据均保存在您的本地设备上，您的数据安全性取决于您对设备的管理。我们建议您不要在公共设备上存储敏感的条码配置信息。

5. 政策更新

我们可能会不时更新本隐私政策。任何更改都将通过在此页面发布新的隐私政策来通知。建议您定期查看此页面。
